# Vif Next-Gen Browser

Vif is a high-performance, AI-integrated browser interface designed for the modern web. It combines minimalist design with powerful smart features to redefine how you interact with information.

## 🚀 Features

- **AI-Integrated Search**: Get instant insights powered by Gemini 3 directly in your search bar.
- **Smart Assistant**: A built-in AI companion always ready to help you analyze pages, translate content, or generate ideas.
- **Next-Gen Workspace**: A high-tech, dark-themed dashboard with glassmorphism effects and neon accents.
- **Shield Protection**: Visual indicators of security and privacy status.
- **Smart Tabs**: Multi-state tab management with layout animations.

## 🛠️ Tech Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion (Motion 12)
- **AI**: Google Gemini API (@google/genai)
- **Icons**: Lucide React

## 📂 Project Structure

- `src/App.tsx`: Main application entry and UI components.
- `src/lib/gemini.ts`: AI service integration.
- `src/index.css`: Custom theme and high-tech styling.

## 🎨 Design Philosophy

Vif follows a "High-Tech Minimalist" aesthetic, utilizing Recipe 5 (Brutalist Creative) and Recipe 7 (Atmospheric Media) from the professional frontend design guidelines. It prioritizes dark mode, high contrast, and fluid motion to create a premium user experience.
